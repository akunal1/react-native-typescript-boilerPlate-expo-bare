# React-Native-BoilerPlate-code

This is ReactNative boiler Plate code with ready to go for production using Expo bare workflow

# Features

1. Typescript
2. Lodash
3. Redux-saga
4. Jest
5. Formik
6. Error boundary
7. ESlint
8. useMemo
9. memo

# Starting the Project

> > yarn install

> > yarn run android

> > > yarn run ios

> > > > yarn run web
